# Professional README Generator
![License: MIT](https://img.shields.io/badge/License-MIT-yellow)

## Description

This project uses node.js to create a Professional README. It simplifies the process of creating a README file. Inquirer and fs were two crucial packages that made the Professional README Generator possible.

## Table of Contents

* [Usage](#usage)
* [Credits](#credits)

## Usage

In VS Code, the user will need to open their integrated terminal. In the terminal, they will need to type node index js. The user will then be prompted with questions regarding each section that should go in a professional README.md.
